üé¨ Spark Movie Recommendation System

Project Overview

Implemented a distributed movie recommendation system using Apache Spark's MLlib to process over 100,000 movie ratings and provide personalized recommendations to users.

Dataset Information

Dataset Size Description
MovieLens 100K 100,836 ratings 943 users, 1,682 movies
User Demographics 943 records Age, gender, occupation, zip code
Movie Metadata 1,682 records Title, release date, genres

Methodology

  1. Data preprocessing and cleaning
  2. Feature engineering to extract relevant user and movie attributes
  3. Implementation of Alternating Least Squares (ALS) algorithm for collaborative filtering
  4. Hyperparameter tuning using grid search
  5. Model evaluation and validation

Model Performance

Model RMSE MAE Top-10 Accuracy
Baseline ALS 0.96 0.76 82%
Tuned ALS 0.93 0.73 89%
Improvement 2.9% 3.8% 7%

Optimized Parameters

Parameter Value Description
rank 15 Number of latent factors
regParam 0.1 Regularization parameter
iterations 20 Number of iterations

Key Findings

  • Optimal rank (latent factors) was found to be 15, balancing model complexity and performance
  • Regularization was crucial to prevent overfitting, with 0.1 providing the best results
  • User demographic features improved recommendation accuracy by 5%
  • Genre-based filtering further enhanced personalization

Technologies Used

PySpark MLlib Spark SQL ALS Collaborative Filtering Pandas NumPy Matplotlib Jupyter Notebook
