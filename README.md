## 🛠️ 技术栈

- **前端**：纯静态 HTML5, CSS3, JavaScript (ES6+)
- **CSS架构**：CSS Grid, Flexbox, CSS自定义属性（变量）
- **设计系统**：米色调色板（Data Poetics设计风格）
- **动画**：CSS Animation + Intersection Observer
- **图表**：Plotly.js（待实现）, Chart.js（已集成）
- **代码高亮**：Prism.js
- **图标**：Lucide Icons（通过CDN）
- **字体**：Google Fonts（Merriweather, Inter）
- **部署**：GitHub Pages（计划）


## 🎨 设计特色

### 米色调色板（Data Poetics设计系统）
- **背景色**：#FAF7F2 (奶油色) / #E8DFD3 (米色)
- **文字色**：#2C2416 (深褐) / #8B7355 (棕色)
- **边框色**：#D4C4B0 (棕褐色)
- **强调色**：
  - #FF6B6B (珊瑚色) - 主要强调
  - #95B99F (鼠尾草绿) - 成功/增长
  - #5B8FB9 (天蓝色) - 链接/信息
  - #D4A574 (金色) - 特殊强调
