# 个人作品集网站

一个现代化的数据分析师与全栈开发者作品集网站，采用响应式设计和现代化 UI 效果。

## 特点

- 🎨 现代化 UI 设计
- 🌓 深色/浅色主题切换
- 🌐 中英文双语支持
- 📱 完全响应式布局
- ✨ 流畅的页面过渡动画
- 🎯 交互式技能展示
- 🖥️ 项目展示组合

## 技术栈

- HTML5
- CSS3 (动画, Flexbox, Grid)
- JavaScript (原生)
- GSAP (动画库)
