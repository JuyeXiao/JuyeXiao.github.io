# Design System 2.0
## 数据分析师作品集设计风格指南 - 米色版

**版本：** v2.2  
**更新日期：** 2025年1月  
**适用范围：** 个人作品集网站设计与开发

---

## 1. 设计理念



**设计哲学：**
- **温暖极简** - 米色的温润质感传达人文关怀
- **理性与感性平衡** - 数据的严谨性与视觉的艺术性相融合
- **渐进式展示** - 信息如呼吸般自然展现
- **空间的诗意** - 用留白创造节奏和焦点

### 1.2 设计原则

| 原则 | 描述 | 实现方式 |
|------|------|----------|
| **层级清晰** | 信息重要性一目了然 | 大胆的字体对比、清晰的视觉层级 |
| **动静结合** | 静态布局配合动态细节 | 微动效、滚动动画、数据渐现 |
| **克制用色** | 95%中性色+5%强调色 | 米色系为主，彩色仅用于关键信息 |
| **网格秩序** | 严格的网格创造和谐 | 12列网格系统，黄金比例分割 |
| **故事叙述** | 每个项目都是一个故事 | 渐进式信息展示，引导式浏览 |

## 3. 布局系统

### 3.1 网格系统

```
容器宽度：
- 最大宽度：1400px
- 内边距：32px（桌面端）、16px（移动端）

网格规范：
- 列数：12列
- 间距：32px（桌面端）、16px（移动端）
- 黄金比例：重要分割采用 1:1.618
```

### 3.2 核心布局模式

#### A. 分屏式布局（Hero Section）
```
┌─────────────────┬──────────────────────┐
│                 │                      │
│   左侧 (1)      │      右侧 (1.618)    │
│   文字信息      │      视觉展示         │
│   cream bg      │      beige bg        │
└─────────────────┴──────────────────────┘

配色方案：
- 左侧背景：cream
- 右侧背景：beige (或渐变)
- 文字：dark / charcoal
- 装饰元素：tan (线条)
```

#### B. Bento Box布局（项目展示）
```
┌──────────────────────┬──────────┐
│    Large (8)         │ Med (4)  │
│    warm-gradient     │ beige    │
├──────────┬───────────┼──────────┤
│ Small(4) │ Small (4) │ Small(4) │
│ cream    │ tan       │ beige    │
├──────────┴───────────┴──────────┤
│         Wide (12)                │
│         accent-gradient          │
└──────────────────────────────────┘

配色策略：
- 主要项目：渐变背景或深色背景
- 次要项目：米色系背景
- 强调项目：强调色渐变（克制使用）
- 边框：统一使用 tan
```

#### C. 杂志式布局（关于页面）
```
┌──────────────┬────────────────────┐
│              │                    │
│   图片区域    │      文字区域       │
│    (40%)     │       (60%)        │
│              │                    │
└──────────────┴────────────────────┘

特点：
- 大图配文字
- 不对称平衡
- 增强阅读体验
```

#### D. 卡片网格布局（技能展示）
```
┌────┬────┬────┬────┬────┐
│ 01 │ 02 │ 03 │ 04 │ 05 │
├────┼────┼────┼────┼────┤
│ 06 │ 07 │ 08 │ 09 │ 10 │
└────┴────┴────┴────┴────┘

特点：
- 等宽网格
- 2px间隔线
- 悬停缩放效果
```

## 5. 交互规范

### 5.1 动画系统

```css
/* 动画时长 */
--duration-fast: 200ms;
--duration-normal: 300ms;
--duration-slow: 600ms;

/* 缓动函数 */
--ease-out: cubic-bezier(0.0, 0, 0.2, 1);
--ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
--ease-elastic: cubic-bezier(0.68, -0.55, 0.265, 1.55);
```

### 5.2 交互模式

| 交互类型 | 触发条件 | 效果 | 时长 |
|---------|---------|------|------|
| **悬停** | 鼠标悬停 | 上移2-4px + 阴影 | 300ms |
| **点击** | 鼠标点击 | 缩放95% | 100ms |
| **滚动渐现** | 进入视口 | 淡入 + 上移20px | 600ms |
| **数字动画** | 进入视口 | 从0计数到目标值 | 2000ms |
| **页面过渡** | 路由切换 | 淡出淡入 | 400ms |

### 5.3 滚动行为

```javascript
/* 渐进展示设置 */
IntersectionObserver配置：
- threshold: 0.1 (10%可见时触发)
- rootMargin: '0px 0px -50px 0px' (提前50px触发)

/* 视差效果（可选） */
- Hero图片：speed = 0.5
- 背景装饰：speed = 0.3
- 前景内容：speed = 1
```
